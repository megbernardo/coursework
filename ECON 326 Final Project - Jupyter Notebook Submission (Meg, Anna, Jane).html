Because we use a linear probability model, we are going to run into heteroskedasticity as an issue.

We can formally run the White Test to diagnose this, although we already know the outcome.

White Test:

$\hat{\epsilon}_{i}^{2} = \delta_{0} + \delta_{1}\hat{Y}_{1} + \delta_{2}\hat{Y}_{1}^{2} + \epsilon_i$
